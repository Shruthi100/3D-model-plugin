Add 3D models easily using Base URL and Model names.

== Description ==

This plugin allows you to add 3D models and view them easily in your wordpress blog.

Upload the model's base URL and the model's name using the related submenus on the admin menu and 
easily select the required model using the ADD 3D MODEL on the posts page.

GLB models are supported.

== Installation ==

1. Install this plugin from the wordpress plugin directory as usual and activate it.
2. The POINT105 AR prompt appears on admin menu, 
   and it contains a list of submenus : Library , Add New Model and Base URL.
3. Set the Base URL and Add the model names using the Add New Model submenu.
4. The list of added models can be viewed at the Library submenu.
5. The models can be selected using the ADD 3D MODEL buttton on the posts page.
6. As soon as the model is selected, the corresponding shortcode appears on the Description box.
